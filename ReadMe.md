# 專案摘要

學習及練習使用 Python 套件(sqlalchemy)，對 PostgreSQL 資料庫，進行資料存／取。

 - raw_sql_crud.py: 純粹以 Raw SQL Statement 進行資料處理。
  
 - sql_express_crud.py:　使用 SQL Expression Language 進行資料處理。 　 

 - orm_crud.py: 使用 ORM 進行資料處理。
